# 📄 HTML Resume Project

## 📌 About This Project

This is a simple **Resume Web Page** created using basic HTML.
It displays my personal details, education, experience, projects, and skills in a structured format.

---

## 🛠️ Technologies Used

* HTML5

---

## 📂 Features

* Header with name and contact details
* Career objective section
* Education details
* Internship experience
* Certifications list
* Projects description
* Technologies and skills

---

## ▶️ How to Run

1. Download or clone this project
2. Open the file in any browser

   ```
   index.html
   ```
3. You will see my resume webpage

---

## 📷 Output

This project shows a simple resume layout using HTML tags like:

* `<header>`
* `<nav>`
* `<h1> to <h2>`
* `<p>`
* `<ul>`

---

## 🎯 Purpose

This project is created to practice:

* Basic HTML structure
* Text formatting
* Lists and sections
* Building a simple webpage
